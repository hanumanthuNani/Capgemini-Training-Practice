## üß† Key Concepts

### Java Program Structure
- **Definition:** Every Java program must have a class and a main method
- **Usage:** Entry point for execution
- **Example:**
```java
public class MyProgram {
    public static void main(String[] args) {
        // Your code here
    }
}
```

### Data Types
- **Definition:** Java has primitive types (int, double, boolean, char) and reference types (String, Arrays, Objects)
- **Usage:** Declare variables to store different kinds of data
- **Example:**
```java
int age = 25;
double salary = 50000.50;
boolean isActive = true;
String name = "John";
```

### System.out.println()
- **Definition:** Method to print output to console
- **Usage:** Display results and debug information
- **Example:**
```java
System.out.println("Hello, World!");
```
